# Carousel Design Patterns

Proven carousel structures for Instagram, LinkedIn, and Facebook. Each pattern includes slide-by-slide layout, text placement guidance, CTA positioning, and visual flow principles.

---

## 1. Narrative Carousel (Storytelling)

**Best for:** Brand stories, case studies, origin stories, customer journeys.
**Platforms:** Instagram, LinkedIn.
**Optimal length:** 7-10 slides.

### Slide Structure

| Slide | Purpose | Content | Visual Treatment |
|---|---|---|---|
| 1 -- Hook | Stop the scroll | Bold question or provocative statement | Full-bleed image or solid brand color. Large centered text (40-60% of slide). |
| 2 -- Context | Set the scene | Background information, "Here's what happened..." | Split layout: image left, text right. Or text over dimmed background. |
| 3-4 -- Rising Action | Build tension | The challenge, obstacle, or problem deepening | Progressive visual intensity. Darker tones, more detail. |
| 5-6 -- Climax | The turning point | Key moment, breakthrough, or insight | Highest visual contrast in the series. Bold accent color. |
| 7-8 -- Resolution | Show the outcome | Results, transformation, data | Lighter tones returning. Charts or before/after imagery. |
| 9 -- Reflection | Extract the lesson | Key takeaway, quote, or moral | Clean design, single message. Brand colors prominent. |
| 10 -- CTA | Drive action | Follow, save, share, visit link | Brand logo, handle, and specific CTA. Arrow or hand pointing icon. |

### Visual Flow
- Use a consistent color gradient across slides (dark to light, or cool to warm).
- Swipe direction indicator: subtle arrow or visual element bleeding off the right edge.
- Maintain consistent text position (always top-left, or always centered).

## 2. Listicle Carousel (Tips / Steps)

**Best for:** Educational content, tips, tools lists, resource roundups.
**Platforms:** Instagram, LinkedIn, Twitter/X.
**Optimal length:** 5-8 slides.

### Slide Structure

| Slide | Purpose | Content | Visual Treatment |
|---|---|---|---|
| 1 -- Title | Hook + promise | "7 Tools That..." or "5 Ways to..." | Number prominent (large, accent color). Topic in clean sans-serif. |
| 2-N -- Items | One item per slide | Number + item title + 1-2 sentence explanation | Consistent layout: number top-left (large, accent color), title centered, brief text below. Icon or small illustration per item. |
| N+1 -- Summary | Recap all items | Numbered list of all items (brief) | Compact grid layout. All items visible. Smaller text. |
| N+2 -- CTA | Drive action | Save this post, follow for more, link in bio | Brand colors, logo, clear directive. |

### Visual Flow
- Number styling must be identical across all item slides (same font, size, color, position).
- Background color can subtly shift per slide within the brand palette.
- Each item slide should be self-contained (readable without other slides).

## 3. Before / After Comparison

**Best for:** Transformations, redesigns, makeovers, process improvements.
**Platforms:** Instagram, LinkedIn, Facebook.
**Optimal length:** 3-7 slides.

### Slide Structure

| Slide | Purpose | Content | Visual Treatment |
|---|---|---|---|
| 1 -- Hook | Promise transformation | "See the difference..." or context title | Split-screen preview. Blurred or partial before/after. |
| 2 -- Before | Show the starting state | "Before" label + image/data/screenshot | Muted tones, desaturated. "Before" label in top-left corner. |
| 3 -- The Process | What changed | Brief list or explanation of changes | Center-aligned text, process icons or numbered steps. Neutral background. |
| 4 -- After | Show the result | "After" label + image/data/screenshot | Saturated, bright. "After" label in top-left corner. Brand accent color highlights. |
| 5 -- Side-by-Side | Direct comparison | Before and After side by side | Split screen, vertical divider line. Labels above each half. |
| 6 -- CTA | Offer the transformation | "Want these results? Link in bio" | Clean brand slide with CTA and social proof (if available). |

### Visual Flow
- Before slides: cooler, muted, slightly desaturated.
- After slides: warmer, vibrant, brand colors prominent.
- Transition slide (process) serves as visual bridge.

## 4. Data Visualization Carousel

**Best for:** Industry stats, survey results, trend reports, annual reviews.
**Platforms:** LinkedIn, Instagram, Twitter/X.
**Optimal length:** 5-8 slides.

### Slide Structure

| Slide | Purpose | Content | Visual Treatment |
|---|---|---|---|
| 1 -- Hook | Headline stat | Single shocking or compelling number | Large number (80% of slide), brief context label. Brand accent color on number. |
| 2 -- Context | Frame the data | Source, methodology, timeframe | Clean text slide. Subtle background pattern. Source citation. |
| 3-6 -- Data Points | One chart/stat per slide | Individual data visualization + insight sentence | Single chart type per slide (bar, line, donut, etc.). Insight text below chart. Brand colors for data fills. |
| 7 -- Synthesis | Connect the dots | Key narrative from all data points | Summary with 3-4 bullet points. Small icons per point. |
| 8 -- CTA | Drive action | Download full report, follow for more | Brand slide with CTA and data source attribution. |

### Visual Flow
- Use the same chart style family across all data slides (all flat, or all 3D, etc.).
- Brand primary color for the main data series; secondary for comparison.
- Consistent axis styling, label fonts, and grid lines.
- Number animations (if video): count-up reveals.

## 5. Quote Carousel

**Best for:** Thought leadership, testimonials, book excerpts, interview highlights.
**Platforms:** Instagram, LinkedIn, Twitter/X.
**Optimal length:** 3-7 slides.

### Slide Structure

| Slide | Purpose | Content | Visual Treatment |
|---|---|---|---|
| 1 -- Context | Introduce the speaker/source | Name, title, topic, or book cover | Photo of person or book cover. Name and credentials below. Brand overlay. |
| 2-5 -- Quotes | One quote per slide | Quote text + attribution | Large quotation marks (accent color, semi-transparent). Quote centered. Attribution bottom-right, smaller text. Solid or gradient background. |
| 6 -- Summary | Key message | The most impactful quote repeated or synthesized | Clean, minimal design. Largest text. Brand accent underline. |
| 7 -- CTA | Drive engagement | "Agree? Share your take below" or "Follow for more" | Brand slide, engagement prompt. |

### Visual Flow
- Consistent quotation mark styling across all quote slides.
- Same background treatment (same color, same texture, same gradient direction).
- Photo of the speaker can appear as a small circular element on each quote slide for continuity.

## 6. Tutorial / Step-by-Step Carousel

**Best for:** How-to guides, recipes, DIY instructions, software walkthroughs.
**Platforms:** Instagram, LinkedIn.
**Optimal length:** 6-10 slides.

### Slide Structure

| Slide | Purpose | Content | Visual Treatment |
|---|---|---|---|
| 1 -- Title | Promise the outcome | "How to [achieve X] in [N] steps" | Finished result image or illustration. Title overlay. Step count badge. |
| 2 -- Prerequisites | What you need | Tools, materials, software, prerequisites list | Clean list with icons. Checkbox-style layout. |
| 3-N -- Steps | One step per slide | Step number + instruction + visual of the step | Step number (large, accent color, top-left). Instruction text (center or bottom). Visual showing the step being performed (photo, screenshot, or illustration). |
| N+1 -- Result | Show the finished product | Final outcome image + congratulatory text | Bright, polished image. "Done!" or "Your turn!" text. |
| N+2 -- CTA | Save and share | "Save for later" + follow prompt | Brand slide. Bookmark icon. Share icon. |

### Visual Flow
- Sequential numbering creates natural reading momentum.
- Progress indicator: subtle bar at bottom that fills across slides, or step dots.
- Consistent camera angle/perspective for each step photo.

## 7. Problem to Solution Carousel

**Best for:** Product launches, service introductions, pain point marketing.
**Platforms:** Instagram, LinkedIn, Facebook.
**Optimal length:** 5-7 slides.

### Slide Structure

| Slide | Purpose | Content | Visual Treatment |
|---|---|---|---|
| 1 -- Hook | Identify with the audience | "Tired of [problem]?" or relatable situation | Frustrated expression imagery or chaotic visual. Muted palette. Bold text. |
| 2 -- Problem Deep Dive | Validate the pain | Specific symptoms, stats, or scenarios | List format with pain point icons (X marks, warning symbols). Darker tones. |
| 3 -- Failed Solutions | Show what doesn't work | Common attempts that fall short | Crossed-out items or "myth vs reality" format. Continued muted tones. |
| 4 -- The Solution | Introduce your answer | Your product/service/method as the solution | Visual shift: brand colors appear. Clean, bright design. Product image or method illustration. |
| 5 -- How It Works | Explain the mechanism | 3-4 key features or steps | Feature icons with brief labels. Clean grid layout. Brand palette. |
| 6 -- Proof | Build credibility | Testimonials, stats, logos, results | Social proof layout: quotes, star ratings, metric callouts. |
| 7 -- CTA | Convert | Clear next step: sign up, buy, book, download | Single CTA button-style element. Urgency element if appropriate. Brand logo. |

### Visual Flow
- Slides 1-3: muted, desaturated, tension-building.
- Slide 4: color shift -- brand palette introduced (the "reveal" moment).
- Slides 5-7: fully branded, confident, polished.

## 8. Product Showcase Carousel

**Best for:** Product launches, feature highlights, collection reveals.
**Platforms:** Instagram, Facebook, LinkedIn.
**Optimal length:** 5-8 slides.

### Slide Structure

| Slide | Purpose | Content | Visual Treatment |
|---|---|---|---|
| 1 -- Hero | Feature the product | Product name + hero image | Clean product photography on brand-colored background. Product name in brand font. |
| 2-3 -- Features | Highlight key features | One feature per slide with visual | Close-up product shots or feature diagrams. Feature name as heading, 1-line benefit below. |
| 4 -- In Context | Show it in use | Lifestyle or usage scenario image | Lifestyle photography. Product in real-world setting. Warm, inviting. |
| 5 -- Specs / Details | Technical or practical info | Key specs, sizes, materials, compatibility | Clean grid or table layout. Icons for each spec. |
| 6 -- Social Proof | Build trust | Reviews, ratings, press mentions | Star ratings, short quote, publication logos. |
| 7 -- Pricing | State the value | Price, bundles, comparison to alternatives | Price prominent. Value framing ("Less than $X/day"). |
| 8 -- CTA | Convert | Shop now, link in bio, limited offer | Urgency-driven design. "Shop Now" CTA. Brand logo and handle. |

### Visual Flow
- Consistent product photography style (same lighting, background, angle family).
- Brand color as background or accent on every slide.
- Price slide should feel premium, not discount (unless that is the brand positioning).

## Universal Carousel Guidelines

### Text Placement Rules
- **Headlines:** Top 25% of slide, centered or left-aligned.
- **Body text:** Middle 50% of slide. Maximum 3 lines per slide.
- **Numbers/Stats:** Large, accent-colored, positioned as focal element.
- **Attribution/Source:** Bottom 10%, smaller, lighter color.
- **CTA text:** Bottom 30%, high contrast, button-style.

### CTA Placement
- Always on the final slide.
- Can also appear subtly on slide 1 (e.g., "Swipe to learn").
- Use action verbs: Save, Share, Follow, Download, Book, Try, Start.
- Arrow icons pointing right (swipe direction) on non-final slides.

### Visual Consistency Across Slides
- Same margin/padding on every slide (recommended: 60px on 1080px canvas).
- Same font pairing across all slides.
- Same logo placement (if present): bottom-center or bottom-right.
- Same background treatment family (all solid, all gradient, all photo-based).
- Slide transition cues: element that bleeds off the right edge to invite swiping.

### Engagement Optimization
- First slide determines 80% of carousel performance -- invest the most effort here.
- Save rate is the strongest signal. Include "Save this for later" CTA.
- Odd number of slides (5, 7, 9) tends to feel more complete than even.
- Last visible slide in preview (slide 1 + partial slide 2) should create curiosity.
