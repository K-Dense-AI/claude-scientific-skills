# Visual Specs by Platform

Complete dimension specifications for every major social media format. All values in pixels unless noted.

---

## Instagram

### Post -- Square
- **Dimensions:** 1080 x 1080 px
- **Aspect Ratio:** 1:1
- **Max File Size:** 30 MB
- **Formats:** JPG, PNG
- **Safe Zone (text):** 960 x 960 px (60px inset from each edge)
- **Notes:** Feed crops to square in grid view. Minimum upload: 320x320.

### Post -- Portrait
- **Dimensions:** 1080 x 1350 px
- **Aspect Ratio:** 4:5
- **Safe Zone (text):** 960 x 1200 px (60px inset from sides, 75px from top/bottom)
- **Notes:** Tallest feed format. Takes up more screen real estate than square.

### Post -- Landscape
- **Dimensions:** 1080 x 566 px
- **Aspect Ratio:** 1.91:1
- **Safe Zone (text):** 960 x 466 px (60px inset from sides, 50px from top/bottom)
- **Notes:** Least engaging format for feed; use sparingly.

### Story
- **Dimensions:** 1080 x 1920 px
- **Aspect Ratio:** 9:16
- **Safe Zone:** 810 x 1420 px (135px from sides, 250px from top, 250px from bottom)
- **Top Bar (username/time):** Top 120px -- do not place critical content
- **Bottom Bar (reply/send):** Bottom 200px -- do not place critical content
- **Sticker Zone:** 250-650px from top (optimal for polls, questions, quizzes)
- **Swipe-Up / Link Zone:** Bottom 200px
- **Notes:** 15-second max per frame. Use vertical orientation only.

### Reel
- **Dimensions:** 1080 x 1920 px
- **Aspect Ratio:** 9:16
- **Safe Zone (for feed preview):** 1080 x 1350 px centered (top 285px and bottom 285px may be cropped in feed)
- **Caption Area:** Bottom 300px -- avoid text overlays here
- **Right Side Icons:** Right 100px in bottom half -- avoid content
- **Notes:** Feed preview crops to 4:5. Cover image should work at both 9:16 and 4:5.

### Carousel
- **Dimensions:** 1080 x 1080 px (square) or 1080 x 1350 px (portrait)
- **Aspect Ratio:** 1:1 or 4:5
- **Max Slides:** 20
- **Safe Zone:** Same as respective post format
- **Swipe Indicator Area:** Bottom-center 200px wide, 50px tall
- **Notes:** First slide aspect ratio sets ratio for all slides. Mix of images and video allowed.

### Profile Photo
- **Dimensions:** 320 x 320 px
- **Display Size:** 110 x 110 px (mobile), 150 x 150 px (desktop)
- **Shape:** Circle crop -- keep subject within 280px diameter centered circle
- **Notes:** Uploaded as square, displayed as circle.

## Facebook

### Post Image
- **Dimensions:** 1200 x 630 px
- **Aspect Ratio:** 1.91:1
- **Max File Size:** 30 MB
- **Safe Zone (text):** 1080 x 530 px (60px from sides, 50px from top/bottom)
- **Text Overlay Rule:** Images with >20% text get reduced distribution (soft guideline)
- **Notes:** Also used for link preview images.

### Story
- **Dimensions:** 1080 x 1920 px
- **Aspect Ratio:** 9:16
- **Safe Zone:** 810 x 1420 px (same as Instagram Story)
- **Top Bar:** Top 150px reserved for profile info
- **Bottom Bar:** Bottom 250px reserved for CTA and reply
- **Notes:** 20-second max per frame. Supports interactive stickers.

### Cover Photo
- **Dimensions:** 820 x 312 px
- **Mobile Display:** Crops to 640 x 312 px (centered)
- **Desktop Display:** Full 820 x 312 px
- **Safe Zone (mobile-safe center):** 640 x 312 px centered (90px bleed each side)
- **Notes:** Does not display on mobile the same as desktop. Keep critical content centered.

### Event Cover
- **Dimensions:** 1920 x 1005 px
- **Aspect Ratio:** 1.91:1
- **Safe Zone:** 1720 x 850 px (100px from sides, 75px from top/bottom)

### Ad Image
- **Dimensions:** 1200 x 628 px
- **Aspect Ratio:** 1.91:1
- **Text Overlay:** Keep below 20% of image area for best delivery
- **Safe Zone:** 1080 x 528 px (60px inset from all edges)
- **Headline Area:** Below image, not in image

### Profile Photo
- **Dimensions:** 180 x 180 px (minimum)
- **Recommended Upload:** 360 x 360 px
- **Display Size:** 176 x 176 px (desktop), 36 x 36 px (comments)
- **Shape:** Circle crop

## Twitter / X

- **Display Size:** 48 x 48 px (timeline), 200 x 200 px (profile)

## YouTube

### Thumbnail
- **Dimensions:** 1280 x 720 px
- **Aspect Ratio:** 16:9
- **Max File Size:** 2 MB
- **Formats:** JPG, PNG, GIF, BMP
- **Safe Zone:** 1152 x 640 px (64px from sides, 40px from top/bottom)
- **Timestamp Overlay:** Bottom-right corner 120 x 30 px -- avoid content here
- **Text Readability:** Ensure text is legible at 120 x 68 px (smallest display size)
- **Notes:** Must have YouTube verification for custom thumbnails. High contrast and faces perform best.

### Channel Art / Banner
- **Dimensions:** 2560 x 1440 px
- **Safe Zone (all devices):** 1546 x 423 px centered
- **Desktop Display:** 2560 x 423 px (center strip)
- **Tablet Display:** 1855 x 423 px (center strip)
- **Mobile Display:** 1546 x 423 px (center strip)
- **TV Display:** Full 2560 x 1440 px
- **Profile Photo Overlap:** Bottom-left on desktop
- **Notes:** Upload at full size. Only the center safe area is guaranteed visible on all devices.

### YouTube Short
- **Dimensions:** 1080 x 1920 px
- **Aspect Ratio:** 9:16
- **Duration:** Up to 60 seconds
- **Safe Zone:** 810 x 1420 px
- **Bottom UI:** Bottom 200px (subscribe button, like/dislike)
- **Right Icons:** Right 100px in bottom half
- **Notes:** Vertical only. Auto-loops.

## Cross-Platform Notes

### DPI and Resolution
- All social platforms use 72 DPI for web display.
- Export at 72 DPI unless print use is also intended.
- sRGB color space for all platforms.

### File Format Recommendations
- **JPG:** Photos, complex images with gradients. Use quality 85-95%.
- **PNG:** Graphics with text, logos, transparency. Larger file size.
- **WebP:** Supported on most platforms. Smaller file size than JPG/PNG.
- **GIF:** Animated content. Limited to 256 colors per frame.
- **MP4:** Video content. H.264 codec, AAC audio.

### Text Overlay Guidelines
- Minimum font size equivalent: 24px at 1080px width (legible on mobile).
- Maximum text coverage: 20-30% of image area for organic reach.
- Always add contrast backing (shadow, stroke, or semi-transparent box) behind text.
- Test readability at 50% zoom to simulate mobile feed viewing.
