# Content Calendar Templates

Reusable calendar templates for different planning horizons and campaign types. Each template uses a consistent column structure suitable for Markdown, CSV, or JSON export.

---

## Standard Column Definitions

| Column | Description |
|--------|------------|
| Date | YYYY-MM-DD format |
| Day | Day of week (Mon-Sun) |
| Platform | Target platform (instagram, facebook, linkedin, tiktok) |
| Format | Post format (reel, carousel, static, story, text, video, document, link) |
| Content Pillar | Thematic category (education, social-proof, behind-the-scenes, promotion, entertainment, community) |
| Hook | Opening line or first 3 seconds concept |
| Visual Type | Image style (photo, graphic, screenshot, UGC, animation, talking-head) |
| CTA | Call-to-action (save, share, comment, link-in-bio, DM, shop-now) |
| Status | Workflow state (idea, draft, review, approved, scheduled, published) |

## Evergreen Content Rotation Calendar

For content that can be recycled on a rolling basis. Assign each piece a rotation frequency.

| Content ID | Platform | Format | Content Pillar | Topic | Rotation Frequency | Last Posted | Next Due |
|-----------|----------|--------|---------------|-------|--------------------|-------------|----------|
| EG-001 | IG | Carousel | Education | [Core topic explainer] | Every 90 days | YYYY-MM-DD | YYYY-MM-DD |
| EG-002 | LI | Text | Social Proof | [Case study summary] | Every 60 days | YYYY-MM-DD | YYYY-MM-DD |
| EG-003 | TT | Short | Education | [Quick tip video] | Every 45 days | YYYY-MM-DD | YYYY-MM-DD |
| EG-004 | IG | Reel | Entertainment | [Relatable industry humor] | Every 60 days | YYYY-MM-DD | YYYY-MM-DD |
| EG-005 | FB | Link | Education | [Blog post reshare] | Every 90 days | YYYY-MM-DD | YYYY-MM-DD |
| EG-006 | LI | Document | Education | [Framework / checklist] | Every 120 days | YYYY-MM-DD | YYYY-MM-DD |
| EG-007 | IG | Static | Promotion | [Signature offer graphic] | Every 30 days | YYYY-MM-DD | YYYY-MM-DD |
| EG-008 | ALL | Carousel | Community | [FAQ answers] | Every 90 days | YYYY-MM-DD | YYYY-MM-DD |

### Evergreen Rotation Rules

1. Never post the same evergreen piece within 30 days of its last appearance on the same platform.
2. Update hooks and visuals on each rotation to keep content fresh.
3. Retire evergreen content after 4 rotations or when engagement drops below 50% of initial performance.
4. Seasonal evergreen content (holiday recipes, summer tips) should be tagged with eligible months.
5. Track each rotation's performance to identify truly evergreen pieces vs. one-time performers.
