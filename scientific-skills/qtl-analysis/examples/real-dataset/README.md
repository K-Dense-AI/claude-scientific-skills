# Optional Real Dataset Catalog

This example provides optional real public datasets for benchmarking and demonstration.

## Run
```bash
python download_real_dataset.py
```

## Output
- `output/real_dataset_catalog.csv`
- `output/download_instructions.txt`

## Note
Synthetic data remains the default path. Real datasets are optional extras.
