# Shiny Dashboard Demo (Python Shiny)

Input:
- Synthetic trait values generated in-app

Process:
- Render interactive trait scatter and table preview
- Expose controls for sample size and reproducibility seed

Output:
- Interactive app at local web URL

Run:
```bash
shiny run --reload app.py
```
