## Hook Selection Guide

| Goal | Best Hook Types | Why |
|------|----------------|-----|
| Maximize shares | Controversy, Hot Take | Polarization drives identity-based sharing |
| Maximize saves | Educational, Framework | Utility content gets bookmarked |
| Maximize comments | Pattern Interrupt, Controversy | Interactive and debate-driven hooks |
| Maximize reach | Curiosity Gap, FOMO | Drives clicks, which signals relevance to algorithms |
| Build trust | Storytelling, Social Proof | Emotional connection + validation |
| Drive conversions | Social Proof, FOMO/Urgency | Proof reduces risk; urgency triggers action |

## Usage Notes

- **Customize aggressively.** Never use a hook template verbatim. The bracketed sections must be filled with specific, concrete details from the brand and topic.
- **Test in batches.** Run 3-5 hook variants per post concept and measure which style resonates with the specific audience.
- **Match hook to format.** Curiosity hooks work best for carousels (swipe to find out). Story hooks work best for video. Educational hooks work best for threads and long-form.
- **Rotate psychology types.** Using the same trigger repeatedly leads to audience fatigue. Alternate between emotional (story, controversy) and rational (educational, social proof) hooks.
- **Front-load the hook.** The first 5-8 words must earn the rest of the sentence. The first sentence must earn the rest of the post.
