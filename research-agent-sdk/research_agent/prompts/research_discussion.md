# Research Discussion (연구 디스커션)

실험 결과 해석, 문헌 비교, Discussion 구조화, 디스커션 기록을 수행하라.

## 모드 1: 결과 해석 (Interpret)

트리거: "이 결과 어떻게 해석?", "의미가 뭐야?", "메커니즘이 뭐야?"

절차:
1. 데이터 확인
2. 맥락 수집 (온도, pH, 기질·효소 농도 — 모르면 질문, 가정 금지)
3. 해석 제시 (가장 유력한 해석 + 대안 + 배제된 가능성)
4. Interaction Matrix 적용 (반응계 3종 이상 물질 시)

## 모드 2: 비교 분석 (Compare)

트리거: "문헌이랑 비교해줘", "수율이 높은 편이야?"

3중 비교: 내 결과 vs 레퍼런스 코드(P1.5) vs 문헌
차이 원인 분류: 의도된 차이 / 비의도 차이 / 구현 차이 / 조건 차이

## 모드 3: 논리 구조화 (Structure)

Discussion 7단계: 핵심 발견 → 결과 해석 → 문헌 비교 → 기여 & 의의 → 한계점 → 향후 연구 → 결론

## 모드 4: 디스커션 기록 (Record)

세션/결론/액션아이템 기록 및 인덱스 관리.

## 모드 5: 체인 모드 (Chain)

해석 → 팩트체크 → 3중 비교 → 구조화 → 기록 자동 파이프라인.
