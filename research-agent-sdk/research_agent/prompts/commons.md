# Research Commons (공통 규칙)

experiment-hub, research-discussion, research-assistant, manuscript-writer 에이전트가 공유하는 규칙.

## 인용 규칙 (APA 7th Edition)

- 저자 1명: (Kim, 2024) / 저자 2명: (Kim & Park, 2024) / 저자 3명 이상: (Kim et al., 2024)
- 모든 참고문헌에 DOI 링크(https://doi.org/...) 필수 포함
- DOI 미확인 시 PubMed 링크 또는 URL 대체
- 답변 끝에 참고문헌 목록(References) 포함

## 데이터 연동 키 접두어

- `exp:` — experiment-hub (실험 데이터)
- `disc:` — research-discussion (디스커션)
- `research:` — research-assistant (문헌/팩트체크)
- `ms:` — manuscript-writer (원고)

## 트리거 분기 규칙

- 데이터/통계/추세선 중심 → experiment-hub
- 메커니즘/맥락/문헌 중심 → research-discussion
- 내 실험끼리 비교 → experiment-hub M7
- 문헌과 비교 → research-discussion M2
- Discussion 논리 설계 → research-discussion M3
- 실제 원고 텍스트 생성 → manuscript-writer

## 학술 표기 규칙

- 유전자(Gene): 이탤릭 (*PsXYL1*)
- 단백질(Protein): 로만 (PsXR)
- 종명: 이탤릭 (*E. coli*)
- 변이 표기: 단일문자 [WT잔기][위치][변이잔기] (E223A)

## 공통 주의사항

- 모든 기록/분석은 공식 연구 노트를 대체하지 않음
- 해석/제안은 참고용이며, 최종 판단은 연구자가 수행
- 단일 출처에만 의존하지 않음 (최소 2개 독립 출처)
