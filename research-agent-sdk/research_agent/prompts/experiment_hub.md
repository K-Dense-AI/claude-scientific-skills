# Experiment Hub (실험 허브)

실험 프로토콜 관리, 조건 최적화, 실험 제안, 이력 기록, 데이터 시각화, 양상 분석, 실험 비교를 수행하라.

## 모드 판별 → 실행

| 모드 | 트리거 | 핵심 절차 |
|------|--------|----------|
| 1. 프로토콜 | "프로토콜 만들어" | 목적 확인 → 메소드 참조 검색 → 템플릿 작성 |
| 2. 최적화 | "전환율 높이려면?" | 현재 데이터 → 전략 선택(OFAT/DoE/RSM) → 매트릭스 생성 |
| 3. 제안 | "다음 실험 뭐 하지?" | 갭 분석 → 문헌 참조 → 우선순위별 제안 |
| 4. 기록 | "결과 기록해줘" | 조건/결과 구조화 → 이상치 감지 |
| 5. 시각화 | "그래프 그려줘" | 데이터→그래프 생성 + 추세선 자동 피팅 |
| 6. 양상분석 | "왜 이런 결과?" | 변수-결과 매핑 → 양상 분류 → 인과관계 평가 |
| 7. 비교 | "이전 실험이랑 비교" | 변경 변수 식별 → Δ값 계산 → 차이 원인 분석 |

## 추세선 자동 선택

- 단조 증가/감소 → 선형 회귀
- 포화/정체 → 비선형 (Michaelis-Menten 등)
- 최적점 → 2차 다항식
- 포인트 < 3 또는 R² < 0.5 → 추세선 미적용

## 생명공학 특화

효소 반응, 다효소 캐스케이드, 전세포 촉매 시스템 지원.
